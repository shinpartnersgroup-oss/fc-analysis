## 1. ETS社のブランド力とプレゼン戦略

- ETS社のブランド力を最大限に活かしたプレゼンテーションを行う方針
- 売上規模・歴史・実績を正確に把握しておく必要がある
- 「ガンマ」製品を保有する兄弟（財閥的存在）が上位にいるとされるが、実態確認が必要
- 韓国ビジネスの特徴として「大きく見せる」傾向があるため、**裏取り（事実確認）が重要**

## 3. フランチャイズ（FC）展開の戦略と契約条件

### 契約構造（2層構造）
| 契約 | 当事者 | 内容 |
|------|--------|------|
| マスターFC契約 | MEGAMI合同会社 ⇔ ETS社/日本法人 | 日本展開の権利 |
| FC契約 | 本部（MEGAMI） ⇔ 各加盟店 | 個別店舗への権利付与 |

### 加盟金の構成
- **現在の加盟店**: 660万円（契約金200万円 + 商材費400万円）
- **FC加盟金**: 400万円（うち200万円がMEGAMIへ、200万円上乗せで400万円で販売）

### 営業権利金について
- 営業担当（末永さん等）に権利金2,500万円を課すことへの異議あり
- 営業コスト（交通費・接待費）を自己負担する立場で権利金は不合理
- → **契約成立時のパーセンテージ制を採用**

### エリア制
- データ不足のため当面は単店出店
- 将来的にエリア一括購入の権利付与を検討

## 4. ロイヤリティの設定

- パーセンテージ制 → 売上が少ない場合の収入不安定リスクあり
- 定額制（1日いくら）→ 安定性あり
- **売上管理システムの導入が不可欠**
  - 予約管理システムで本部が売上をリアルタイム把握
  - 売上の過少申告リスクへの対策としても重要

---

## 5. 契約書の重要事項

| 項目 | 内容 |
|------|------|
| 同業種NG | 情報流出防止のため同業種の加盟を禁止。発覚時は権利剥奪 |
| 加盟金の性質 | 一時金（返金不可）であることを明記。設備費・広告宣伝費は別途負担 |
| 商標・ブランド管理 | 「公式」「日本本部」の表現は使用禁止 → **「正規公認フランチャイズ」に統一** |
| 価格統一 | **定価5万5千円を下回ることを禁止**。値下げ発覚時は権利剥奪。ネット安売りも禁止 |
| 覚書 | 契約書本体とは別に細かいルールを覚書として整備 |
| トラブル対応 | 都度協議。基本は寄り添う姿勢でタイムリーに対処 |
| 供給途絶リスク | 将来的には日本国内製造体制の構築を目指す |

## 6. 加盟店とFCの差別化

### 現在の加盟店の位置付け
- CURAIMの看板は使用不可
- 既存店舗にメニューとして導入する形（メニューフランチャイズ）

### FC限定の差別化メニュー（検討中）
- 微小循環器によるトータルケアメニュー
- ヘッドスパ（ドライヘッドスパ）の技術提供
- 水素吸入器のオプション設置
- ファスティング（断食プログラム）のCURAIMブランド化
- FC限定商品のリリース
- まつげ・眉毛の育毛メニュー

## 7. 商品供給・物流体制

- 韓国からの直輸入は不可 → **代行会社を経由**（手数料約10%）
- MEGAMI社が半年前に前払いで在庫購入（約2,000万円規模）
- FC店舗への発送は**MEGAMI社から直接**（送料効率を考慮）
- **最低発注ロットの設定が必要**（現在1箱単位で非効率 → 最低10箱程度を検討）
- ガンマの製造リードタイム：**発注から6ヶ月**

## 8. 研修体制

### 現状（加盟店向け）
- 2日間の研修のみ → その後は自力

### FC向け研修（大幅に充実化）
| 項目 | 内容 |
|------|------|
| 期間 | 約1週間 |
| 費用 | 約50万円 |
| 内容 | 技術研修 + 接客研修（お辞儀の角度、スリッパの並べ方まで） |
| 主要講師 | 原先生（カワセ先生）：技術指導 / 昌子先生：接客指導（資格保有） |
| 場所 | 本社（モチベーション向上のため） |
| 頻度 | 年1〜2回のバージョンアップ研修を実施 |
| 特典 | 県外研修生には一蘭本店訪問などモチベーション向上施策 |

## 9. スーパーバイザー（SV）制度

- SVが**1ヶ月間**みっちり店舗に入り込んで指導
- SV費用：月額約80万円（売上向上効果で実質プラス）
- 課題：SV人材が限られている（現状は原先生のみ → 同時進行が困難）
- 運用施策：
  - 毎日のZoom報告会
  - 技術動画の提出
  - AIによる録音分析（カウンセリング・クロージングの質向上）

---

## 10. 売上データ・提案資料の整備

- 銀行への借入提案資料に**売上予測値**が必要
  - マックス / ノーマル / ネガティブの3パターンのシミュレーション
- 固定費（日常経費、電気代、水道光熱費等）の算出
- 成功事例のデータ収集 → 大津先生、ショウコさんの店舗に依頼
- 苫小牧市の資料を参考にする
